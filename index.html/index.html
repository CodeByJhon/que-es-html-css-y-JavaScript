¿Qué son HTML, CSS y JavaScript?

HTML, CSS y JavaScript son las tres tecnologías fundamentales para construir páginas web.
HTML (HyperText Markup Language) se encarga de la estructura y el contenido de la página,
definiendo elementos como títulos, párrafos, listas e imágenes. CSS (Cascading Style Sheets)
se utiliza para el diseño y la presentación visual, controlando el color, la tipografía,la disposición
y otros estilos para hacer que la página se vea atractiva.

Por último, JavaScript añade interactividad y funcionalidad, permitiendo crear animaciones, responder
a las acciones del usuario y manejar datos de forma dinámica.

Pilares del Desarrollo Web

        HTML (HyperText Markup Language) 🌐📝

        HTML es el lenguaje de marcado fundamental que estructura el contenido de las páginas web. 

       🏗️ Define los elementos como textos 📄, imágenes 🖼️ y enlaces 🔗, actuando como el "esqueleto" de tu sitio. 
       
       ¡Es la base para construir cualquier página web! 🧱


        CSS (Cascading Style Sheets) ✨🎨

        CSS es el lenguaje que controla la presentación visual de las páginas web. 💅 Se encarga de los colores 🌈, 

        fuentes ✒️, diseño 📐 y estilos, haciendo que la información estructurada por HTML se vea atractiva y profesional.
       
        🌟 ¡Es el "maquillaje" y el "diseño de interiores" de la web! 🏡


        JavaScript (JS) 🚀💡

        JavaScript es un lenguaje de programación que aporta interactividad y dinamismo a las páginas web. 🎮 Permite 

        funcionalidades como animaciones ✨, validación de formularios ✅ y respuestas a acciones del usuario, 

        transformando sitios estáticos en experiencias dinámicas y cautivadoras. 💡 ¡Es el "cerebro" que hace que la web

        cobre vida y se mueva! 🧠⚡

Ejemplo de código HTML

    <!DOCTYPE html>
    <html lang="es">
    <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>Mi Primera Página Web</title>
    </head>
    <body>
        <h1>¡Hola, mundo!</h1>
        <p>Este es un párrafo de ejemplo.</p>
    </body>
    </html>

Explicando el Código

  • <!DOCTYPE html>: Le dice al navegador que el documento es una pagina web escrita en HTML5.

  • <html>: Es el contenedor principal que envuelve toda la pagina web.

  • <head>: Contiene información sobre la pagina que no es visible.

  • <body>: Es donde va todo el contenido que las personas verán.

  • <h1>: Se usa para el titulo mas importante de la página.

  • <p>: Se utiliza para definir un párrafo dentro del texto.
